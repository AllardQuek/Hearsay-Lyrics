## Tracks / Categories

Build a **new** next-generation AI agent that uses multimodal inputs/outputs and goes beyond simple text-in/text-out in one of these three categories. [geminiliveagentchallenge.devpost](https://geminiliveagentchallenge.devpost.com/)

### 1. Live Agents 🗣️

- **Focus:** Real-time interaction (audio/vision) [geminiliveagentchallenge.devpost](https://geminiliveagentchallenge.devpost.com/)
- **Goal:** Build an agent users can talk to naturally and interrupt. [geminiliveagentchallenge.devpost](https://geminiliveagentchallenge.devpost.com/)
- **Example ideas:**  
  - Real-time translator  
  - Vision-enabled tutor that “sees” your homework  
  - Customer support voice agent that handles interruptions gracefully [geminiliveagentchallenge.devpost](https://geminiliveagentchallenge.devpost.com/)
- **Mandatory tech:**  
  - Must use **Gemini Live API** or **ADK (Agent Development Kit)**  
  - Agents must be hosted on **Google Cloud** [geminiliveagentchallenge.devpost](https://geminiliveagentchallenge.devpost.com/)

### 2. Creative Storyteller ✍️

- **Focus:** Multimodal storytelling with interleaved/mixed output [geminiliveagentchallenge.devpost](https://geminiliveagentchallenge.devpost.com/)
- **Goal:** Agent that behaves like a creative director, weaving text, images, audio, and video into a single fluid output stream. [geminiliveagentchallenge.devpost](https://geminiliveagentchallenge.devpost.com/)
- **Example ideas:**  
  - Interactive storybooks (text + generated illustrations inline)  
  - Marketing asset generator (copy + visuals + video in one go)  
  - Educational explainers (narration with diagrams)  
  - Social content creator (caption + image + hashtags together) [geminiliveagentchallenge.devpost](https://geminiliveagentchallenge.devpost.com/)
- **Mandatory tech:**  
  - Must use **Gemini’s interleaved/mixed output** capabilities  
  - Agents must be hosted on **Google Cloud** [geminiliveagentchallenge.devpost](https://geminiliveagentchallenge.devpost.com/)

### 3. UI Navigator ☸️

- **Focus:** Visual UI understanding and interaction [geminiliveagentchallenge.devpost](https://geminiliveagentchallenge.devpost.com/)
- **Goal:** Agent that becomes the user’s hands on screen, observing the browser/device display, interpreting visual elements, and acting based on user intent. [geminiliveagentchallenge.devpost](https://geminiliveagentchallenge.devpost.com/)
- **Example ideas:**  
  - Universal web navigator  
  - Cross-application workflow automator  
  - Visual QA testing agent [geminiliveagentchallenge.devpost](https://geminiliveagentchallenge.devpost.com/)
- **Mandatory tech:**  
  - Must use **Gemini multimodal** to interpret screenshots/screen recordings  
  - Must output **executable actions**  
  - Agents must be hosted on **Google Cloud** [geminiliveagentchallenge.devpost](https://geminiliveagentchallenge.devpost.com/)

## Judging Criteria

Projects will be judged on: [geminiliveagentchallenge.devpost](https://geminiliveagentchallenge.devpost.com/)

1. **Innovation & Multimodal User Experience (40%)**  
   - Does it break the “text box” paradigm?  
   - Does the agent help users “see, hear, and speak” seamlessly?  
   - Does it have a distinct persona/voice?  
   - Does the experience feel live and context-aware vs disjointed and turn-based? [geminiliveagentchallenge.devpost](https://geminiliveagentchallenge.devpost.com/)

2. **Technical Implementation & Agent Architecture (30%)**  
   - Effective use of Google GenAI SDK or ADK  
   - Backend robustly hosted on Google Cloud  
   - Sound agent logic and error handling  
   - Evidence of grounding and reduced hallucinations [geminiliveagentchallenge.devpost](https://geminiliveagentchallenge.devpost.com/)

3. **Demo & Presentation (30%)**  
   - Does the video clearly define the problem and solution?  
   - Is the architecture diagram clear?  
   - Is there visual proof of Cloud deployment?  
   - Does the video show the actual software working? [geminiliveagentchallenge.devpost](https://geminiliveagentchallenge.devpost.com/)
